Robot Factory Simulator
=======================

A graphical simulation of a robotic factory with moving robots.

Requirements:
- Java 17 or higher (JRE)

How to Run:
-----------
Windows: Double-click run.bat or the JAR file
Linux/Mac: Run ./run.sh (make executable first: chmod +x run.sh)
Or: java -jar RobotFactorySimulator.jar

Features:
- Animation menu: Start/Stop robot movement simulation
- File menu: Save/Open factory configurations (.ser files)
- Robots move between machines and charging stations

Controls:
- Animation > Start Animation: Begin simulation
- Animation > Stop Animation: Pause simulation
- File > Save Canvas: Save factory state
- File > Open Canvas: Load saved factory
